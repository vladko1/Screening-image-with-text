# Screening-image-with-text
Windows forms application for Screening with text a given image (groups of pixels become replaced with text).

Welcome to Screening with text preview

This is a desktop application, which does screening with text based on some input image. The application is suitable for processing photos and images with higher resolution. Processing of images with lower resolution is not achieved satisfactory results.
